# HIRA/공공데이터 원천 파일

아래 파일명으로 데이터를 넣으면 `backend/scripts/import_hira_data.py`가 자동으로 찾아 DB에 적재합니다.
현재 저장소에는 일부 파일이 `*.csv.csv` 형태로 들어와 있어도 stem 기준으로 탐색합니다.

- `age_cause.csv`: 난임시술유형별 연령별 원인별 현황
  - 사용 위치: 난임 시술 상담 준비 > 연령대 및 원인별 시술 현황
- `process.csv`: 난임시술유형별 연령별 시술과정별 현황
  - 사용 위치: 난임 시술 상담 준비 > HIRA 과정별 타임라인
- `attempt.csv`: 난임시술유형별 시술횟수/차수 현황
  - 사용 위치: 난임 시술 상담 준비 > 반복 시술 구간 확인
- `institution.csv`: 난임시술유형별 의료기관종별 현황
  - 사용 위치: 병원·약국 찾기 > 의료기관종별 분포 참고
  - 참고: 현재 파일은 구형 `.xls`라 `xlrd`가 없으면 적재가 건너뛰어질 수 있습니다.
- `places.csv`: 정부 지정 체외수정 의료기관 지정현황
  - 사용 위치: 병원·약국 찾기 > 체외수정 지정 의료기관 예시, 지역별/종별 분포
  - 주요 컬럼: `연번`, `지역`, `요양기관명칭`, `종별`, `주소`, `전화번호`
- `ing.csv`: 국민건강보험공단 연도별 난임시술 현황
  - 사용 위치: 난임 시술 상담 준비 > 연도별 난임시술 이용 추이
  - 주요 컬럼: `진료년도`, `진료인원(명)`, `진료건수(건)`, `1인당건수(건)`

```bash
python scripts/inspect_csv_columns.py
```
